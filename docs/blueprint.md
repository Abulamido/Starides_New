# **App Name**: Starides Unified Marketplace

## Core Features:

- Unified Landing Page: A single landing page with role-based login/registration and a global search.
- Customer App: Browse products, place orders, track deliveries, and manage wallet balance. Payments will happen with Paystack
- Vendor App: Manage products, inventory, accept/reject orders, and track earnings.
- Rider App: Accept order assignments, share live location, and confirm deliveries. Location managed using Google Maps API.
- Admin Dashboard: Manage users, vendors, products, orders, payments, and analytics.
- AI-Powered Product Recommendations: Use machine learning tools to generate personalized product recommendations for customers based on browsing history and purchase patterns.
- Firebase Integration: Utilize Firebase Auth, Firestore, Realtime Database, Storage, Cloud Functions, and Cloud Messaging.

## Style Guidelines:

- Primary color:#5d8dd4 to capture a sophisticated, modern feel from the logo and brand identity.
- Background color: A very light grayish blue (#F0F4F8) that provides a clean and unobtrusive backdrop.
- Accent color: A muted grayish violet (#A09FC7), which offers contrast and visual interest without overpowering the interface.
- Body and headline font: 'Inter' sans-serif for a clean, modern, neutral look.
- Minimalist line icons that are easy to understand and visually consistent across all platforms.
- A clean, responsive grid layout that adapts to different screen sizes and devices.
- Subtle transitions and animations (using Framer Motion) to enhance user experience.
